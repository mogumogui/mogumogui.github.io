    卸载MicrosoftSQLServer2008R2安装程序

    无法卸载，因为安装了 

    SQL Server 2008 R2 Database Engine Shared
SQL Server 2008 R2 Common Files

    但是在已安装软件中找不到这两个软件，那么解决办法是

    首先点击运行，输入cmd，进入命令行模式 然后输入WMIC

    然后执行product list 找到

    SQL Server 2008 R2 Database Engine Shared
SQL Server 2008 R2 Common Files

    二者后面分别对应：

    {36F70DEE-1EBF-4707-AFA2-E035EEAEBAA1}

    {A2122A9C-A699-4365-ADF8-68FEAC125D61}

    然后进入windos/system32目录下

    执行：

    c:\Windows\System32>msiexec.exe /x {36F70DEE-1EBF-4707-AFA2-E035EEAEBAA1}
c:\Windows\System32>msiexec.exe /x {A2122A9C-A699-4365-ADF8-68FEAC125D61}

    然后再卸载

    2019年10月07日 （记）

    Posted on 2019-10-07 Edited on 2020-10-10

    《 what for ? 》 -> 独听好评！ 

    第一次从汽车站走到火车站（18分钟），戴着耳机听着歌一路边看边走，虽穿梭在人潮中，却感受不到外界的嘈杂，仿佛一个人安静的走在乡间小路上，虽独处却一点也不孤独！

    LinKin Park 《In The End》 

    it starts with one thing, i don't know why!To explain in due time!


之前没有在这样的环境下听过这首歌，但今天伴着微风，袖子也跟着欢快起来！把我带到了一个熟悉的陌生地方，那里有成堆成团的波波草，有着同样的微风，有着我所期待的软石路，在那我可以沿着路慢慢悠悠地散着步，看着道路两旁向我寒暄的绿色精灵，还有空中飘拂的蒲公英，多么美妙，多么神奇，多么让人流连忘返的佳境。

突然之间，歌声转瞬即逝，背后的阴影把我吸回现实，有那么一瞬间令我手足无措，当我清醒过来，眼眶莫名润湿，耳旁却依旧是Chester·Bennington和Mike·Shinoda的声音，我下意识地按着太阳穴回想，脑海里的画面却是那么的零零散散，我明白那可能是我今后为之奋斗的东西！我大声喊加油！自己！这一刻声带没有颤抖！

检完票进站后，找了个空位置坐下来，刚下来就被椅子爆按一顿，这时耳机已不再是老熟人的歌，换成了没有词的《whatfor？》，不知是今天的我感性还是，还是今天的歌尤其动人！每一首让我都能有感触！！！

今天扫了一个聋哑人的码给他转了二十块，我知道这样的影响会更不好，但是我的手但是忍不住拿出手机去扫那个码，之后跟他发了一些鼓励他的话，不知道他能不能看到，不论真假，还是希望社会上不论是残疾人或是假装残疾人乞讨的人，都能好好活下去！

    本机WIFI密码查询

    Posted on 2019-10-06 Edited on 2020-10-09
    获取本地连接过的WIFI密码

    创建一个.bat文件输入以下：

    @echo off

    set /p i=””

    netsh wlan show profiles name d= %i% key= clear

    pause > nul


    也可以直接输入在CMD命令行下输入 netsh wlan show profiles name = “WIFI NAME” key = clear
